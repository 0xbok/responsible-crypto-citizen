# WIP: Be a Responsible Crypto Citizen
See and vote on governance proposals if you hold the related governance token.

It lists the governance proposals from Aave, Compound, Uniswap using subgraphs from _The Graph_. See the demo here: https://gmridul.github.io/responsible-crypto-citizen. This will reflect the current status of the project.

## Next milestone
Add voting capability on these proposals directly from this web app.
